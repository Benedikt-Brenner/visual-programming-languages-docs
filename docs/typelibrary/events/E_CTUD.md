### E\_CTUD

Event-Driven Up-Down Counter

![](https://user-images.githubusercontent.com/113907528/204895474-3f88876a-7ce5-406e-8f44-765c1b97226c.png)

*   Input
    *   CU Count Up 
    *   CD Count Down
    *   R Reset
    *   LD Load
    *   PV Preset Value
*   Output
    *   CO Count Output Event
    *   RO Reset Output Event
    *   QU CV   >= PV
    *   QD CV  \<= 0
