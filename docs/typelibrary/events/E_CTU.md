### E\_CTU

Event driven up counter

![](https://user-images.githubusercontent.com/113907528/204894537-54f1c9c9-de57-4beb-ad38-697fff012b4b.png)

*   Input
    *   CU Count up
    *   R Reset
    *   PV Process Value
*   Output
    *   CUO Count up output
    *   RO Reset output
    *   Q True if CV >= PV
    *   CV Counter Value
